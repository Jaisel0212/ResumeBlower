### End-to-End Recruitment Automation Platform

ResumeBlower is a **Python-based recruitment automation tool** designed to simplify and accelerate the hiring process by automating resume handling, candidate analysis, and recruiter collaboration.

It helps recruiters save time, reduce manual effort, and make data-driven hiring decisions.

---

## üîç Problem Statement

Recruiters spend a significant amount of time:
- Downloading resumes from emails
- Manually scanning resumes
- Searching candidates based on skills and experience
- Comparing candidates and matching them with job descriptions

**ResumeBlower automates the entire recruitment workflow in one platform.**

---

## ‚ú® Key Features

1. **Processing Any Number of Resumes**  
   - Handles bulk resume processing efficiently without manual intervention.

2. **Resume Information Extraction**  
   - Automatically extracts:
     - Contact Details
     - Total Experience
     - Qualifications
     - Skills and other key information  
   - Supports both **PDF and DOCX** formats.

3. **Resume Repository Management**  
   - Stores resumes in a centralized repository.
   - Enables quick searching and accessing of resumes when required.

4. **Candidate Search & Filtering**  
   - Search candidates based on:
     - Name
     - Skill
     - Experience
   - Helps shortlist candidates faster.

5. **Automatic Resume Downloading from Inbox**  
   - Downloads resumes directly from the email inbox based on defined criteria.
   - Eliminates manual downloading of attachments.

6. **Resume Analysis Using Derived Metrics**  
   - Analyzes resumes using calculated metrics such as:
     - Total experience
     - Skill relevance
     - Qualification alignment

7. **Candidate Comparison**  
   - Compare any two candidates side by side.
   - Helps recruiters make informed hiring decisions.

8. **JD Matching & Candidate Scoring**  
   - Matches resumes against Job Descriptions.
   - Generates a relevance score for each candidate.
   - Simplifies shortlisting for specific roles.

9. **Recruiter Collaboration & Chat**  
   - Enables chatting with other recruiters or HR professionals.
   - Supports sharing candidate information and files within the platform.

10. **Chatbot Assistant**  
    - Built-in chatbot assistant to guide users.
    - Provides help and answers queries related to the recruitment process.
